## Назначение

Обновить `agsekit` в том же Python-окружении, где запущена текущая CLI.

## Команда

```bash
agsekit pip-upgrade
```

## Что делает

1. Проверяет, что `agsekit` установлен через `pip` в текущем окружении.
2. Считывает текущую установленную версию через `pip show agsekit`.
3. Выполняет `pip install agsekit --upgrade`.
4. Повторно считывает версию и печатает итог:
   - если версия изменилась, сообщает старую и новую версии;
   - если версия не изменилась, сообщает, что уже установлена последняя версия.

## Пример

```bash
agsekit pip-upgrade
```
