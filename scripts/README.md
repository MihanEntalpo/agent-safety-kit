# Различные технчиеские скрипты agsekit

## install.sh - универсальный скрипт установки для `curl | bash`

Работает в:

* Linux
  * debian
  * arch
* MacOS
* WSL (windows linux subsystem)

Что делает:

* Создаёт venv
* Устанавливает в него agsekit
* Добавляет agsekit в PATH для запуска откуда угодно
